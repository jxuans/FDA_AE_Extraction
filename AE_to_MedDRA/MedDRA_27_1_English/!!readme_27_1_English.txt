The following is a list of files in the distribution Internet File Download for MedDRA® 

Version 27.1

MedDRA Terminology and SMQ Files: 

The MedDRA files are delivered in extended ASCII file format for English and the Western European language versions. Other languages use UTF-8 format. These files may be viewed in a browser or relational database. The MSSO provides a desktop browser at no additional cost to subscribers; it can be downloaded by navigating to:  https://www.meddra.org/browsers. In addition, there is an online web-based browser located at:  https://tools.meddra.org/wbb/ and a browser suited for mobile devices at mmb.meddra.org. A MedDRA ID and password are required to access these browsers. 

Please see the MedDRA Distribution File Format Document for specific file structures and relationships. The document defines the MedDRA file structure and how the files should be linked.
